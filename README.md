🕰️ TimeTravel Agency — L'Excellence Temporelle

"Le futur appartient à ceux qui visitent le passé." — Bienvenue dans l'interface de gestion de la TimeTravel Agency, leader mondial du tourisme temporel de luxe.

🚀 Liens du Projet

Webapp en direct : https://timetravel-agency-eta.vercel.app/#destinations

Dépôt Source : https://github.com/ynsnait-lab/timetravel-agency

✨ Fonctionnalités Clés

🎭 Expérience Immersive (UI/UX)

Design "Dark Gold" : Une interface premium utilisant le Glassmorphism et des accents dorés pour une immersion totale.

Hero Cinématique : Fond vidéo dynamique (Paris 1889) pour une entrée en matière spectaculaire.

Galerie Interactive : Cartes de destinations (Paris, Crétacé, Florence) avec effets de survol et chargement optimisé (Lazy Loading).

🤖 Intelligence Artificielle & Personnalisation

L'Oracle (Chatbot IA) : Un concierge virtuel intelligent capable de conseiller les voyageurs sur les époques, les tarifs (en Chronos) et la sécurité temporelle.

Analyseur de Résonance (Quiz) : Un algorithme de recommandation qui identifie l'époque idéale du voyageur en fonction de son profil psychologique.

📅 Gestion des Flux

Module de Réservation : Formulaire de validation pour bloquer un créneau dans la Timeline sans risquer de paradoxe temporel.

🛠️ Stack Technique

Front-end : HTML5, CSS3 (Tailwind CSS pour le design atomique), JavaScript Vanilla.

Animations : Transitions fluides et micro-interactions via CSS & JS.

Icônes : FontAwesome 6.0 & Lucide Icons.

Hébergement : Vercel (Déploiement continu depuis GitHub).

🤖 Transparence IA (Vibe Coding)

Ce projet a été réalisé en utilisant des outils d'IA générative pour accélérer le cycle de développement :

Conception & Code : Logique et structure optimisées par Gemini (Google).

Génération de Visuels : Images et vidéos cinématiques créées via Midjourney, Runway et Gemini lors de la phase 1.

Prompt Engineering : Utilisation de prompts système rigoureux pour définir la personnalité de "L'Oracle".
